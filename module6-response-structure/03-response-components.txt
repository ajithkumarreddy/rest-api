Common Express response methods:

/* Send */
res.send() - sends a response body, such as text, HTML, or a buffer.
res.json() - sends a JSON response.

/* Status */
res.status() - sets the HTTP status code.
res.sendStatus() - sets the status code and sends the response.

/* Redirect */
res.redirect() - redirects the client to another URL.

/* Files */
res.sendFile() - sends a file to the client.
res.download() - forces a file download in the browser.
res.render() - renders a view using a template engine such as EJS.

/* Cookie */
res.cookie() - sets a cookie in the browser.
res.clearCookie() - clears a cookie.

/* Headers */
res.set() - sets one or more headers.
res.header() - sets a response header.
res.type() - sets the Content-Type header.

/* Location */
res.location() - sets the Location response header.

/* End */
res.end() - ends the response without sending a body.
