Common request properties in Express:

req.method - the HTTP verb, such as GET, POST, PUT, or DELETE.
req.url - the full URL path including the query string, such as /users/12?sort=name.
req.path - the URL path without the query string, such as /users/12.
req.get("Authorization") - reads a specific request header, such as Authorization or User-Agent.
req.ip - returns the client IP address.
req.host - returns the host value, such as api.example.com.
